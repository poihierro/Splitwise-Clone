<h1>SplitWise - Clone</h1>

<hr />
<h4>Splitwise clone app is a free tool for friends and roommates to track bills and other shared expenses, so that everyone gets paid back.</h4>

<H2>#Tech stacks used</h2>
<ul>
    <li>Kotlin</li>
    <li>Java</li>
    <li>MVVM</li>
    <li>Room Database</li>
    <li>Retrofit</li>
</ul>
